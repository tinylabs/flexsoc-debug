# flexsoc-debug
JTAG/SWD debugger implemented via FPGA

![bd](.images/flexdebug.png)

## Dependencies
- libftdi1  
  `sudo apt install libftdi1-dev`
- verilator  
   `sudo apt install verilator`
- fusesoc  
   `pip3 install --upgrade --user fusesoc`
- vivado  
  `https://www.xilinx.com/support/download.html`
